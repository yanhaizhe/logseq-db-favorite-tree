# 功能清单

本文档用于汇总 `Logseq DB Favorite Tree` 当前已实现能力、交互边界与建议中的下一步迭代方向，便于后续排期、验收与沟通。

## 1. 产品目标

- 将 Logseq 收藏页转化为可持续维护的层级导航树
- 让页面组织结构依赖页面属性而非手工目录
- 提供轻量、常驻、低打扰的悬浮导航入口
- 在不同 graph 下分别保留各自的使用上下文

## 2. 当前已实现功能

### 2.1 根节点与关系解析

- 以 Logseq 收藏页作为树根节点来源
- 通过页面属性关系构建树层级，默认属性名为 `parent`
- 支持节点属性、多值属性、页面引用对象等多种属性值结构
- 对 DB 内部属性键格式做兼容匹配，而不只依赖字面 `parent`
- 支持循环依赖检测，避免树递归进入死循环

### 2.2 树浏览与定位

- 根节点默认按页面标题排序
- 子节点默认按页面标题排序
- 支持展开单个节点并懒加载子节点
- 支持一键展开全部 / 折叠全部
- 支持当前打开页面高亮
- 支持一键定位当前页
- 定位时会展开所有命中的相关路径，而不是只展开第一条
- 定位节点会显示定位标记并短暂高亮
- 当用户切换到其他页面后，旧的定位标记会自动清除
- 顶部显示当前页面在树中的包屑路径，并支持点击跳转

### 2.3 搜索与筛选

- 支持树内搜索
- 搜索结果保留命中节点及其祖先路径
- 搜索状态下隐藏无关分支，减少视觉噪音
- 搜索结果支持关键词高亮
- 搜索状态下支持手动折叠 / 展开结果树
- 搜索时禁用拖拽排序，避免筛选态与真实层级顺序混淆

### 2.4 显示模式与界面形态

- 支持 `sidebar`、`floating`、`mixed` 三种显示模式
- 默认显示模式预设为 `sidebar`
- `mixed` 模式下初始化优先展示侧边栏
- 固定 `sidebar` 或 `floating` 时隐藏模式切换按钮
- 原生侧边栏与悬浮 UI 互斥显示，避免重复渲染
- 支持从侧边栏切回悬浮面板，也支持从悬浮面板切到原生侧边栏

### 2.5 排序与自定义顺序

- 树目录默认使用标题排序作为基础顺序
- 支持同级节点拖拽自定义排序
- 自定义排序建立在默认排序之上，新节点仍能获得合理默认位置
- 根节点排序和子节点排序均支持自定义覆盖
- 排序状态按 graph 维度独立持久化

### 2.6 面板与悬浮交互

- 支持工具栏图标打开 / 关闭面板
- 面板支持自由拖动
- 面板右下角支持拖拽缩放
- 面板支持一键恢复默认尺寸
- 面板标题栏支持双击收回为悬浮球
- 收回后可通过悬浮球再次展开面板
- 悬浮球支持自由拖动
- 悬浮球拖拽结束后自动吸附到左右边缘
- 面板初始位置支持左 / 右侧偏好
- 拖动后优先使用用户最近一次位置

### 2.7 面板内容区与工具区

- 面板功能区支持一键收起 / 展开
- 功能区收起后为树内容腾出更多可视空间
- 标题栏左上角说明文本已收进 `i` 图标提示中
- `i` 图标支持悬停或点击后查看说明
- 标题栏提示区域与拖拽区域做了事件隔离，避免误拖拽
- 原生侧边栏模式保留搜索、定位、展开全部 / 折叠全部、刷新与设置
- 面板与侧边栏按钮图标已做更强语义化区分
- 常用按钮支持 tooltip 瞬时提示
- 面板主体、工具条与包屑区域使用轻量滚动条样式

### 2.8 刷新与状态恢复

- 支持手动刷新
- 支持自动刷新
- 支持面板内暂停 / 恢复自动刷新
- 支持 DB 变化监听配合轮询兜底
- 自动刷新间隔可配置
- 记住展开状态、滚动位置、上次定位节点
- 不同 graph 独立记住面板位置、尺寸、悬浮球位置、展开状态、功能区状态、排序、显示模式与视图模式

### 2.9 主题与兼容性

- 自动跟随 Logseq 明暗主题
- 面向 DB graph 使用场景
- 已声明不支持 file graph
- 对旧版或差异化返回结构做尽量归一化处理

## 3. 当前交互边界

- 层级关系只来自页面属性，不会修改 Logseq 原始数据结构
- 拖拽排序仅限同级节点内重排，不支持跨层级改父子关系
- 搜索状态下不允许拖拽排序
- 当前不提供从插件内直接“新增收藏”或“移出收藏”的操作
- 当前不提供“恢复默认排序”的显式按钮
- 当前定位虽支持多路径展开，但没有“上一条 / 下一条命中路径”导航

## 4. 建议补充的下一阶段功能

### 4.1 P0：高价值低风险

- 恢复默认排序
- 清除当前层自定义排序
- 用视觉状态标记当前层是否处于自定义排序
- 拖拽时增加更明显的落点提示与排序反馈

### 4.2 P1：提升日常使用频率

- 从当前页一键加入收藏
- 从节点快捷移出收藏
- 节点右侧增加轻量操作菜单
- 增加“只展开当前路径”或“折叠其他分支”

### 4.3 P2：增强复杂场景体验

- 多命中路径切换导航
- 搜索结果高亮路径
- 更细的异常提示与空状态说明
- 针对大树场景做进一步性能优化
